# Tiff Stacker and Processer

Dies ist der Code der asl Anhang zu meiner BA Arbeit gegeben war. 
Leider ist die asufürung nicht möglich da ich die Daten dafür nicht herraus geben darf.
Der Grobe verlauf des Programmes ist der folgende:

+ Es werden bilder vom typ .dm3 gelesen und in TIFF convertiert
+ Es werden einzelne TIFF bilder gelesen und um ein Hellen Punkt geschnitten und gestacked
+ Es werden wichtige Daten und Rechnungen gemacht die in txt dateien abgespeichert werden.
+ Die Daten werden Visualisiert
+ Files werden züruk in .dm3 konvertiert für zukünftige verarbeitung

Im Ordener Output sind Beispiel Ergebnisse zu sehen.
